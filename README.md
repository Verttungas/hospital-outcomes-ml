# hospital-outcomes-ml

Proyecto final de Machine Learning para predecir el tipo de egreso hospitalario en datos de la SSA.

## Objetivo

Predecir si un individuo egresa por mejoría o por defunción usando metodología CRISP-ML.

## Estructura

- `data/`: datos del proyecto.
- `docs/`: documento final, rúbrica y referencias.
- `notebooks/`: análisis exploratorio y pruebas.
- `R/`: funciones reutilizables.
- `scripts/`: scripts ejecutables.
- `models/`: modelos entrenados, si aplica.

## Flujo de trabajo

1. Crear rama desde `main`.
2. Hacer cambios.
3. Push a la rama.
4. Abrir Pull Request.
5. Esperar revisión de Copilot y de un compañero.
6. Resolver conversaciones.
7. Merge mediante squash.
